# dasboard-blog

# REST API
List of user routes:

| Route                 | HTTP    | Description         |  
| --------------------- | ------- | ------------------- |
| /users                | GET     | Get all the users   |
| /users/:id            | GET     | Get a single user   |
| /users                | POST    | Create user         |
| /users/:id            | PUT     | Update user         |
| /users/:id            | DELETE  | Delete user         |

## Usage
with only npm:

```
 - client
$ npm install
$ npm run dev

  - server
  $ npm install
  $ npm run dev


```
